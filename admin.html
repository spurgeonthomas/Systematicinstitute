What's new in v2

  1. Version History — every save creates a snapshot. Restore any previous version with one click.

How to get your GitHub download URLs

  1. Go to github.com/spurgeonthomas/Systematicinstitute
  2. Click Add file → Upload files and upload your PDF or XLSX
  3. Raw URL format: https://raw.githubusercontent.com/spurgeonthomas/Systematicinstitute/main/YOUR-FILE.pdf
  4. Paste into the Download URL field below, then click Save All
