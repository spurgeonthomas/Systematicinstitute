How to update your live site

  1. Pick a section in the sidebar (Hero, Value Props, Timeline, etc.)
  2. Edit the fields — content is pre-filled from your current site
  3. Click Save Changes — your live site updates instantly, no redeploy needed
  4. For artifact files: upload to GitHub first, then paste the raw URL in the Artifacts section
Quick Start Checklist
Upload artifact files to GitHub — go to github.com/spurgeonthomas/Systematicinstitute → Add file → Upload
Format: https://raw.githubusercontent.com/spurgeonthomas/Systematicinstitute/main/YOUR-FILE.pdf
Change your admin password in Settings before publishing
Default: Systematic2026! — stored in your browser localStorage
Save each section once to push defaults into Firestore, then edit from there
Tip: open each panel, click Save — this seeds the database with your current content

How to get your GitHub download URLs

  1. Go to github.com/spurgeonthomas/Systematicinstitute
  2. Click Add file → Upload files and upload your PDF or XLSX
  3. Raw URL format: https://raw.githubusercontent.com/spurgeonthomas/Systematicinstitute/main/YOUR-FILE.pdf
  4. Paste into the Download URL field below, then click Save All
